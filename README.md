# CropImageView

**Google+'s image cropping style**

## Usage

- `setImageBitmap(Bitmap bitmap)`
- `getCroppedBitmap(int width, int height, Bitmap.Config config)`
- `rotate(int degrees)`

## Caveat

No support for landscape orientation yet!
